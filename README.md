# 💳 Stripe Payment Gateway Integration

## 📌 Project Overview
This project is a Stripe Payment Gateway Integration built using **React.js, Node.js, Express.js, and Stripe API** as part of the **Growfinix Technology Internship – Task 3**.

## 🚀 Features
- Secure Stripe Checkout Integration
- Payment Success Page
- Payment Cancel Page
- React Frontend
- Node.js & Express Backend
- Stripe API Integration

## 🛠️ Tech Stack
### Frontend
- React.js
- Vite
- CSS

### Backend
- Node.js
- Express.js
- Stripe API

## 📂 Project Structure

```text
Stripe-Project/
│
├── frontend/

└── README.md
```

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/amritashrivastva/Growfinix-Stripe-Payment-Gateway.git
```

### Install Frontend

```bash
cd frontend
npm install
npm run dev
```

### Install Backend

```bash
cd backend
npm install
npm start
```

## 🔐 Environment Variables

Create a `.env` file inside the `backend` folder.

```env
STRIPE_SECRET_KEY=your_stripe_secret_key
```

> Do not upload your `.env` file to GitHub.
